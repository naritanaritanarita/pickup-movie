# プランニング規約

## 計画ファイルに必須の項目

### 担当エージェント（必ず明記）

実装を担当するエージェントを明記すること。

```markdown
## 担当エージェント
- バックエンド: laravel-developer
- フロントエンド: typescript-developer
```

**選択基準**:
- PHP/Laravel → laravel-developer
- TypeScript/フロントエンド → typescript-developer
- 両方必要な場合 → 実装順序と共に両方を記載

### その他の必須項目

- **実装ファイル**: 新規作成・修正するファイルの絶対パス
- **実装順序**: 番号付きリストで手順を明記
- **重要な考慮事項**: セキュリティ、パフォーマンス、制約など
